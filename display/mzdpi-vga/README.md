# Display drivers for 2.8 inch Touch Screen for Raspberry Pi

Fast install
```
sudo curl https://raw.githubusercontent.com/paulloft/raspberry/master/display/mzdpi-vga/mzdpi-vga-autoinstall-online | sudo bash -s
````
